Reflexión: Implementando Accesibilidad y Usabilidad

Tecnología

En mi portafolio utilicé Bootstrap para el diseño inicial, aprovechando sus componentes predefinidos como carouseles, offcanvas y spinners. En esta reflexión, estoy utilizando Tailwind CSS, una herramienta de utilidad que permite un diseño más controlado y flexible.

Ambas herramientas son excelentes, pero Tailwind me permite un enfoque más personalizado, mientras que Bootstrap agiliza los desarrollos con sus componentes listos para usar.

Adicionalmente, Tailwind ofrece una excelente integración con frameworks modernos como React y Vue, lo que facilita su uso en proyectos de mayor complejidad.

Accesibilidad y Usabilidad

En mi portafolio implementé diversas técnicas de accesibilidad para garantizar que todos los usuarios puedan interactuar con el contenido de manera eficiente:

  • Navegación por teclado mejorada: 
    button:focus, a:focus, input:focus {
    outline: 3px solid #ff6347; /* Establece un contorno visible */
    outline-offset: 2px;
}
  • Compatibilidad con tecnologías asistivas: Uso de etiquetas ARIA, por ejemplo: 
    <button aria-label="Cerrar menú de accesibilidad" id="closeBanner" class="close-btn">×</button>
  • Alternativas textuales para imágenes: Descripciones en el atributo alt. 
    <img src="path/to/ctrl.png" alt="Icono de Ctrl"></img>
  • Texto redimensionable: Uso de unidades relativas como em, rem o % para los tamaños de fuente.
  • Subtítulos y transcripciones: Añadí subtítulos a videos usando el atributo track. 
    <video controls>
    <source src="movie.mp4" type="video/mp4">
    <track src="subtitles_en.vtt" kind="subtitles" srclang="en" label="English"></track>
</video>
  • Diseño responsivo: Implementé @media queries para que mi portafolio funcione bien en dispositivos móviles.

Además, añadí un apartado de accesibilidad que proporciona atajos a las diferentes secciones del portafolio. Al iniciar, un offcanvas muestra indicaciones de dónde consultar los atajos y cómo navegar eficientemente por el sitio.

Conclusión

Desarrollar este portafolio fue todo un reto. Aprendí a implementar y combinar múltiples herramientas como carouseles, offcanvas, videos y spinners. Incluso integré un video editado con CapCut en la página de inicio, lo cual me llevó tiempo entender, pero finalmente logré dominarlo.

Uno de los mayores desafíos fue resolver un error donde el offcanvas se escondía bajo el main, que solucioné moviendo el HTML fuera del contenedor principal.

Para futuras mejoras, me gustaría implementar un formulario de contacto completamente funcional, simplificar el diseño con un enfoque minimalista y agregar más apartados al portafolio.

Además, planeo explorar otras herramientas como SASS para gestionar mejor los estilos y frameworks de JavaScript como Next.js para optimizar el rendimiento y la organización del código. Esto permitirá una mayor escalabilidad y un diseño aún más profesional.
